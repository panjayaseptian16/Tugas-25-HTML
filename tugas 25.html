Mengenal Semantic Elements Di HTML 5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel yang ada di website kita, tag article bisa kita gunakan dengan element headder dan footer. Karena tag article memiliki type block, artinya element ini memliki lebar sesuai browser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel. Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
